MonoProxy Privacy Policy

Effective Date: July 2026

Overview

MonoProxy is a local proxy server application that supports HTTP / HTTPS together with SOCKS4 / SOCKS4a / SOCKS5 on the same port, and includes a built-in proxy self-test. We take your privacy seriously. This privacy policy explains how we handle your data.

Data Collection

Information We Don't Collect

  • ❌ Personal identity information (name, email, phone, etc.)
  • ❌ Device information
  • ❌ Location information
  • ❌ Usage behavior data
  • ❌ Web browsing history

Information We Process

  • ✅ Network traffic data processed locally on device
  • ✅ Temporary runtime statistics (displayed only in app)
  • ✅ User settings (saved only on device locally)

Data Usage

Local Processing

  • All data is processed only on your device
  • No data is uploaded to any servers
  • No data is shared with third parties

Network Functions

  • The proxy service only forwards HTTP / HTTPS and SOCKS4 / SOCKS4a / SOCKS5 requests locally
  • Does not record or store access content
  • Does not analyze network traffic
  • If SOCKS5 username/password authentication is enabled, credentials are stored only on the device, never uploaded, and never written to any log
  • The built-in proxy self-test only makes a local connection to a public test host (example.com) to verify the proxy works; it does not involve any of your personal data

Data Security

Local Storage

  • App data is protected by iOS standard security mechanisms
  • Other apps cannot access our data
  • Data is automatically encrypted when device is locked

Network Transmission

  • Uses standard HTTPS/TLS encryption
  • No personal information is transmitted

Third-Party Services

This app does not use any third-party analytics services, advertising services, or data collection services.

Data Retention

  • All data exists only temporarily while the app is running
  • All data is automatically deleted when the app is uninstalled
  • Users can clear data at any time within the app

Children's Privacy

This app does not collect any personal information, making it suitable for users of all ages.

International Users

This app can be used globally. Data processing is not restricted by geography because all data is processed locally on the user's device.

Your Rights

  • Access Right: You can view all data in the app at any time
  • Deletion Right: You can delete the app or clear data at any time
  • Modification Right: You can change app settings at any time
  • Restriction Right: You can stop using the proxy service at any time
