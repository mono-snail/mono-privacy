MonoProxy 隐私政策

生效日期：2026年7月

概述

MonoProxy 是一款本地代理服务器应用，在同一端口上同时支持 HTTP / HTTPS 以及 SOCKS4 / SOCKS4a / SOCKS5 协议，并内置代理自测功能。我们非常重视您的隐私。本隐私政策说明了我们如何处理您的数据。

数据收集

我们不收集的信息

  • ❌ 个人身份信息（姓名、邮箱、电话等）
  • ❌ 设备信息
  • ❌ 位置信息
  • ❌ 使用习惯数据
  • ❌ 网络浏览历史

我们处理的数据

  • ✅ 仅在设备本地处理的网络流量数据
  • ✅ 临时存储的运行统计数据（仅在应用内显示）
  • ✅ 用户设置（仅保存在设备本地）

数据使用

本地处理

  • 所有数据仅在您的设备上处理
  • 不会上传到任何服务器
  • 不会与第三方共享

网络功能

  • 代理服务仅在本机转发 HTTP / HTTPS 及 SOCKS4 / SOCKS4a / SOCKS5 请求
  • 不会记录或存储访问内容
  • 不会分析网络流量
  • 若启用 SOCKS5 账号密码验证，凭据仅保存在设备本地，绝不上传，也不会输出到任何日志
  • 内置代理自测功能仅向公开测试地址（example.com）发起本地连接以验证代理可用性，不涉及您的个人数据

数据安全

本地存储

  • 应用数据使用 iOS 标准安全机制保护
  • 其他应用无法访问我们的数据
  • 设备锁定时数据自动加密

网络传输

  • 使用标准 HTTPS/TLS 加密
  • 不传输任何个人信息

第三方服务

本应用不使用任何第三方分析服务、广告服务或数据收集服务。

数据保留

  • 所有数据仅在应用运行时临时存在
  • 卸载应用时所有数据自动清除
  • 用户可随时在应用内清除数据

儿童隐私

本应用不收集任何个人信息，因此特别适合所有年龄段用户使用。

国际用户

本应用可在全球使用，数据处理不受地理位置限制，因为所有数据都在用户设备本地处理。

您的权利

  • 访问权：您可以随时查看应用内的所有数据
  • 删除权：您可以随时删除应用或清除数据
  • 修改权：您可以随时更改应用设置
  • 限制权：您可以随时停止使用代理服务
